# Security Policy

AlertaVida is a real-time disaster alert system. Security is taken seriously.

## Reporting a Vulnerability

Please report vulnerabilities **privately** via GitHub Security Advisories:

1. Go to the [Security tab](https://github.com/CaioOlivieri/AlertaVida/security)
2. Click **Report a vulnerability**
3. Provide a detailed description and steps to reproduce

Do **not** open public issues for security vulnerabilities.

## Response window

You can expect an initial response within **7 days** of submission.
